# WellCall Lite

Простое приложение для видеозвонков peer-to-peer с WebRTC.

## Возможности

- Видеозвонки между двумя людьми
- Несколько участников (grid-вид)
- Экранное демонстрация с аудио
- Переключение камеры (мобильные)
- Зашифрованный чат (E2E)
- Передача прав на звонок
- Индивидуальная громкость для каждого участника
- QR-код для быстрого подключения

## Использование

1. Откройте `index.html` в браузере
2. Нажмите "Создать комнату" для нового звонка
3. Поделитесь ссылкой или QR-кодом с собеседником
4. Собеседник откроет ссылку и присоединится

## Требования

- Современный браузер с поддержкой WebRTC
- Камера и микрофон
- HTTPS или localhost (required for WebRTC)

## Структура

```
wellcall-lite/
├── index.html          # Главная страница
├── css/
│   ├── main.css    # Стили главной страницы
│   └── call-ui.css # Стили интерфейса звонка
└── js/
    ├── main.js         # Приложение
    ├── signaling.js   # WebSocket сигналинг
    ├── webrtc-handler.js # WebRTC логика
    ├── media-handler.js  # Камера/микрофон/экран
    ├── crypto.js     # Шифрование (ECDH + AES-GCM)
    ├── uuid.js     # Генератор ID
    ├── ice-servers.js # STUN/TURN серверы
    ├── call-ui.js  # UI компонент
    └── volume-manager.js # Управление громкостью
```

## Технологии

- Pure JavaScript (ES Modules)
- WebRTC для медиа
- WebSocket для сигналинга
- Web Crypto API для E2E шифрования
- QRCode.js (CDN) для QR-кодов
